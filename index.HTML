Glosario Informático
A-M

A B C D E F G H I J K L M N-Z ➡️

A

Acceso Directo: Icono especial que representa a una carpeta o un archivo y que abre o ejecuta ésta de una forma cómoda y rápida.

Acceso telefónico a redes: Aplicación que permite utilizar una línea telefónica y un módem para conectarse a otro equipo.

Active Desktop: Escritorio activo de Windows 98 que permite tener contenido activo en el Escritorio y poner como fondo o papel tapiz una página Web, un vídeo, un sonido, etc.

Active X: Tecnología de Microsoft que proporciona contenido multimedia (interactivo) a las páginas Web.

B

Barra de canales: Opción del Escritorio que proporciona una lista de los canales disponibles en su equipo.

Barra de herramientas: Conjunto de botones que representan las opciones de menú más comunes o las utilizadas con más frecuencia.

Barra del explorador: Panel que se abre en la parte izquierda de las ventanas.

Beta: Versión anterior a la Alfa y que puede ser la versión definitiva que se comercializará en un determinado tiempo.

C

Caché: Carpeta o memoria intermedia que almacena temporalmente los archivos del equipo.

Carpeta: Contenedor que sirve para almacenar archivos u otras carpetas.

Compilador: Programa que traduce código fuente a lenguaje máquina.

CD-ROM:: Disco compacto de sólo lectura, soporte de almacenamiento de datos de gran capacidad

D

Driver: Software que permite al sistema operativo interactuar con hardware.

Disco óptico: Disco que usa la tecnología láser para el registro y lectura de la información.

Dirección IP: Cadena numérica que identifica a una máquina en una red IP.

Disco Duro: Dispositivo de almacenamiento de datos no volátil.

E

Ejecutable:Dícese del archivo que puede poner en marcha un programa.

Extensión: Sufijo de un archivo que indica su formato (.txt, .jpg).

Ethernet: Tecnología para redes cableadas.

e-mail: Nombre inglés que designa el correo electrónico.

F

Fingering: Medio de reunir información sobre un usuario de la red a partir de su nombre, dirección de correo, etc.

Firmware: Software integrado en hardware.

Fibra Óptica: Medio de transmisión de datos usando luz.

Formatear: Acción de dar formato o estructura a un disco o disquete en forma de sectores, pistas y clúster. Esta acción borra la información que contuviese el disco.

G

GPU: Unidad de Procesamiento Gráfico.

Gigabyte: Unidad de almacenamiento (aproximadamente 1,000 megabytes).

GUI: Interfaz Gráfica de Usuario.

GPS: Sistema de Posicionamiento Global.

H

Hi8: Estándar de grabación creado para videocámaras analógicas.

HTTP: Protocolo de transferencia de hipertexto.

Hardware: Componentes físicos de una computadora.

Hacker: Informáticos que utilizan sus grandes conocimientos para traspasar cualquier barrera informática.

I

IP: Protocolo de Internet, dirección única de un dispositivo en red.

Importar: Transferir o enviar ficheros a otro programa distinto del que los generó..

Inteligencia Artificial: Simulación de inteligencia humana en máquinas.

Interfaz: Aspecto que presentan los programas tras su ejecución mediante el cual ejercemos la comunicación con éstos

J

JavaScript: Lenguaje de programación para desarrollo web.

Java: Lenguaje de programación orientado a objetos.

Joystick: Dispositivo de entrada para juegos.

Jumper:Enlace o puente que se activa dentro de una placa base o tarjeta mediante una palanca diminuta.

K

KHz: Medida para determinar la cantidad de muestras por segundo que se tiene de un sonido y su calidad.

Kbps: Kilobits por segundo, medida de velocidad de transferencia.

Keylogger: Software malicioso que registra pulsaciones de teclado.

Kilobyte: Unidad de almacenamiento (1,024 bytes).

L

LBA: Modo especial de direccionamiento del disco duro con el que se puede acceder a particiones de más de 528 Mbytes.

Linux: Sistema operativo de código abierto.

Latencia: Tiempo de respuesta en una red.

Lenguaje de Programación: Conjunto de instrucciones para crear software.

M

Maestro: Disco que funciona como principal en aquellos ordenadores que utilizan más de un disco duro

Mapa de bits: Tipo de archivo gráfico (compuesto por puntos) de imágenes para PC.

Megabyte: Unidad de almacenamiento (1,024 kilobytes).

Megabyte : Medida que equivale a 1.000 Kb, aproximadamente un millón de caracteres.
